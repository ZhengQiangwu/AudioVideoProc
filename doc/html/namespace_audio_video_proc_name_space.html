◆ GetPrivData()

AUDIOVIDEOPROC_API const char * AudioVideoProcNameSpace::GetPrivData ( int ModuleNum,
char * Key )

获取编码器指定键下的私有属性值

当前的代码默认设置是

privDataMap["b-pyramid"] = "none";

privDataMap["preset"] = "superfast";

privDataMap["tune"] = "zerolatency";

可设置的其余值，举出的部分例子是

"b-pyramid": 用于控制 B 帧金字塔的设置。

——"none": 禁用 B 帧金字塔。

——"strict" : 严格的 B 帧金字塔设置。

"preset" : 用于设置预设编码器参数，以平衡编码速度和质量。

——"ultrafast" : 极快编码速度，质量较低。

——"superfast" : 超快编码速度，质量稍低。

——"fast" : 快速编码速度，质量适中。

——"medium" : 中等编码速度，质量适中。

——"slow" : 较慢编码速度，较高质量。

"tune" : 用于调整编码器以适应特定场景。

——"zerolatency" : 低延迟模式。

——"film" : 适用于电影场景的设置。

——"animation" : 适用于动画场景的设置。

——"grain" : 适用于带有颗粒噪声的场景。

——"psnr" : 优化 PSNR（峰值信噪比）。

"x264opts" : 用于 x264 编码器的设置。

——"subme" : 子像素运动估计设置。

——"me_range" : 运动估计搜索范围设置。

"x265-params" : 用于 x265 编码器的设置。

——"aq-mode" : 适应性量化模式。

"qp" : 用于指定量化参数（QP，Quantization Parameter）。

参数
ModuleNum模块序号
Key键
返回
对应的值
